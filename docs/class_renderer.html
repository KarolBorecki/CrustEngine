Detailed Description

Class used to render simple shapes on the window.

Class held by RenderWindow.hpp for drawing purposes. It is able to render all shapes that is needed to show mesh on the flat screen. It is able to render renderable objects on the screen with given lightning. Refer to RenderWindow class for more information as this class itself is more low level.

See also
RenderWindow.hpp

Draws given mesh on the screen.

Class held by RenderWindow.hpp for drawing purposes. It is able to render all shapes that is needed to show mesh on the flat screen. It also considers lightning and object's postion (In future also rotation).

Parameters
meshMesh that will be drawn.
posPosition in 3D space at which the mesh will be drawn.
rotRotation in 3D space at which the mesh will be drawn.
camCamera, from which perspective the mesh will be projected.
lightDirlightning direction on the scene. Right now Crust engine only supports 1 directional light on the scene //TODO
projectLightFlag if the light should be proojected.
