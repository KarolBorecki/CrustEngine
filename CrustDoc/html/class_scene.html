Detailed Description

Class holding objects rendered on window and camera, from which the projection will be calculated.

This class is passed to render window as the information what to render and how to render. The RenderWindow loads scene and calcuates the projection matrix based on mainCamera values. At each frame RenderWindow goes through renderObjects and draws it.
