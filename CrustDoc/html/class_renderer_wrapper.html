◆ Init()

RendererWrapper::RendererStatus RendererWrapper::Init ( )
inline

Initzializs graphic wrapper. Should be called on wrapper start.

Runs SDL Init funciton. Must be called before any drawing will be done on new Renderer. Also after this call the RendererWrapper::CreateWindow method must be called in order to create new window.

Returns
STATUS_OK if everyting is fine and STATUS_FAIL is something failed and graphic interface cannot start.
