\doxysubsection{Detailed Description}
Class used to render simple shapes on the window. 

Class held by \mbox{\hyperlink{_render_window_8hpp}{Render\+Window.\+hpp}} for drawing purposes. It is able to render all shapes that is needed to show mesh on the flat screen. It is able to render renderable objects on the screen with given lightning. Refer to \mbox{\hyperlink{class_render_window}{Render\+Window}} class for more information as this class itself is more low level.
