\doxysubsection{Detailed Description}
Class holding objects rendered on window and camera, from which the projection will be calculated. 

This class is passed to render window as the information what to render and how to render. The \mbox{\hyperlink{class_render_window}{Render\+Window}} loads scene and calcuates the projection matrix based on main\+Camera values. At each frame \mbox{\hyperlink{class_render_window}{Render\+Window}} goes through render\+Objects and draws it.
