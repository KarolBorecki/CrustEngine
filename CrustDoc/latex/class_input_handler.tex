{\footnotesize\ttfamily void Input\+Handler\+::\+Poll\+Event (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [static]}}



Method that should be called once per frame. This function checks for any pending events. 

This method checks if any event occured and if so sets last\+Event to this event. Then You are able to check last event using \mbox{\hyperlink{class_input_handler_aa90d8a2336b6a041e0dfdc25a1645d6b}{Get\+Last\+Event}} method. Without this function called last\+Event will be set to UNKNOWN state, but \mbox{\hyperlink{class_input_handler_a80edcd5f1a346e8a17b16ed9579b2a23}{Is\+Pressed}} method shall work fine without it.
