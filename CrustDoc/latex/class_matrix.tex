The returned string is in a following format\+:~\newline
 First line contains information about the size of the matrix as follows\+: \mbox{\hyperlink{class_matrix}{Matrix}} of size\+: \texorpdfstring{$<$}{<}rows\texorpdfstring{$>$}{>}x\texorpdfstring{$<$}{<}columns\texorpdfstring{$>$}{>}.~\newline
 Next lines contains the matrix in the correct order of rows and columns.
