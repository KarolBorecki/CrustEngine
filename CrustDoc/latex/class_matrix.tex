\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Matrix$<$ T $>$}
Represents the mathematical matrix and is able to preform matrix calculations. 

This matrix class assumes that x value is the number of a column and y value is the number of a row. This means that matrix\mbox{[}3\mbox{]}\mbox{[}2\mbox{]} is value in 3rd row and 2nd column. Also height says how many rows there are and width says how many columns there are.
